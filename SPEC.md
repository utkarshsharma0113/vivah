# Vivah — Indian Matrimonial Platform

## Brand
- Primary: #8B1A4A (rose)
- Accent: #C9952A (gold)
- Background: #FDF6EE (ivory)
- Text: #1A1A2E

## Pages
1. Homepage — hero search, featured profiles, how it works, plans, footer
2. Register — 3-step form
3. Search — filters + profile grid
4. Profile detail — full view, blurred for non-premium
5. Dashboard — matches, interests, chat, membership
6. Admin — user management, photo moderation, revenue

## India-specific profile fields
religion, caste, subCaste, gotra, manglik (YES/NO/DOESNT_MATTER),
motherTongue, height, complexion, education, occupation,
annualIncome, familyType, familyValues, nriStatus, kundliUrl

## Membership Plans
- Free: browse only
- Silver: ₹999/month — send 10 interests
- Gold: ₹1999/month — unlimited interests + chat
- Platinum: ₹3999/month — gold + verified badge + spotlight

## Business Rules
- Chat unlocks only after mutual interest
- Photos blurred for free users
- Kundli visible only to Gold/Platinum
- Age must be 18+
- Phone OTP required at registration
